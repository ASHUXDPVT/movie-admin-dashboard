# 🎬 Movie Booking Admin Dashboard

A modern React Admin Dashboard for managing movie bookings, users, and ticket workflows.

---

## 🚀 Features

- 📊 Dashboard with analytics charts (Recharts)
- 📅 Calendar scheduling (react-big-calendar)
- 🧩 Kanban board (Drag & Drop ticket management)
- 👥 User management (CRUD + search + pagination)
- 🌙 Dark / Light mode
- ⚡ Smooth animations (Framer Motion)
- 🎨 Responsive UI with Tailwind CSS

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- React Router DOM
- Context API (State Management)
- Recharts (Charts)
- react-big-calendar
- @hello-pangea/dnd
- Framer Motion
